A Python GUI-based Smart Travel Planner application that suggests trip types based on user budget, destination, and number of days.

## Features
- Enter destination
- Select number of days
- Enter total budget
- Get smart trip suggestions:
  - Low Budget Trip
  - Comfortable Trip
  - Luxury Trip

## Technologies Used
- Python
- Tkinter (GUI)

## How to Run
1. Install Python
2. Run the file:
   python smart_trip_advisor.py

## Output
The app suggests the best trip plan based on your budget.
